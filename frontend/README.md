# How to run

```
# Clone this repository
$ git clone https://github.com/vinnycosta9898/ayth-system-jwt

# Go to the directory
$ cd auth-system-jwt

# Install Dependencies
$ npm install
$ yarn dev
$ pnpm install

# Run Web Server
$ npm run dev
$ yarn dev
$ pnpm run dev
```
